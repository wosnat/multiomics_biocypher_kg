# Paper Section Texts: Status and Plan

Standalone methods texts and supporting materials that can be assembled into whatever paper we decide to write. Each piece lives in `docs/` as markdown. Tables and figures are generated by scripts that pull live data (Neo4j or cached JSON), so they auto-update after KG rebuilds.

| # | Piece | File | Status | Notes |
|---|-------|------|--------|-------|
| 1 | Three-tier gene ID mapping | `docs/methods_gene_id_mapping.md` | **Done** | Includes cross-assembly bridging (EZ55, MIT1002 Diamond-based protein matching), row-level resolution, diagnostics. Recently expanded with case studies. |
| 2 | Position-based annotation merge | `docs/methods_position_fallback_merge.md` | **Done** | MIT9313 case study, coordinate matching algorithm, results table, test coverage. |
| 3 | Functional annotation as graph edges | `docs/methods_functional_annotation.md` | **Done** | GO ancestry closure, EC hierarchy, KEGG 4-level hierarchy, COG categories, Cyanorak Roles, TIGR Roles. Covers annotation sources (Cyanorak vs eggNOG-mapper vs UniProt), dual representation rationale, properties-only decisions (Pfam, COG OGs). |
| 4 | Omics adapter + paperconfig system | `docs/methods_omics_adapter.md` | **Done** | Paperconfig YAML schema, edge source determination (EnvironmentalCondition vs OrganismTaxon), publication metadata extraction, edge properties, significance cascade, pre-resolved CSV integration, multi-paper aggregation, edge identity/parallel edges, data validation. |
