**Organism(s):** *Prochlorococcus* MED4 and MIT9313 (axenic vesicle vs whole-cell shotgun proteomics + MED4 vesicle DNA-seq)

## Notes

- The Mascot identification probability is a peptide-ID quality score, not a quantitative abundance; `rankable: false`.
- Vesicle/cell compartment is captured via the Experiment.compartment property; this paper provided the precedent for the convention reused in Biller 2022.
- Table S4 is `top_n` scope: 1079 of 1949 MED4 ORFs had >=1 read but only the top 50 by avg coverage are reported. Ranking among 50 is meaningful; absence is not non-detection.
